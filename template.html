What is the purpose of this code?

public List getThem() {
  List list1 = new ArrayList(); 
  for (int[] x : theList) {
    if (x[0] == 4) {
      list1.add(x);
    }
  }
  return list1; 
}
  • What kinds of things are in theList?
  • What is the significance of the zeroth subscript of an item in theList?
  • What is the significance of the value 4?
  • How would I use the list being returned?

Avoid Disinformation

  • Spelling is information
  • Inconsistent spellings is disinformation.
  • Be aware of automatic code completion
  • the developer is likely to pick an object by name without seeing your copious comments or even the list of methods supplied by that class
  • names for very similar things sort together alphabetically

Make Meaningful Distinctions

Don't write code solely to satisfy a compiler or interpreter.

... by misspelling:

Class klass;
Class clazz;

If names must be different, then they should also mean something different.

public static void copyChars(char[] a1, char[] a2) { 
  for (int i = 0; i < a1.length; i++) {
    a2[i] = a1[i]; 
  }
}
public static void copyChars(char[] source, char[] destination) { 
  for (int i = 0; i < source.length; i++) {
    destination[i] = source[i]; 
  }
}
